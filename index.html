Settlement Calculator (Terzaghi) Python
# 1D consolidation settlement — Terzaghi method def primary_settlement(Cc, e0, H, sigma0, delta_sigma): """ Cc: Compression index e0: Initial void ratio H: Layer thickness (m) """ Sc = (Cc / (1 + e0)) * H * np.log10( (sigma0 + delta_sigma) / sigma0 ) return Sc # Time factor for degree of consolidation def time_factor(U): if U <= 0.6: return (np.pi / 4) * U**2
Consolidation settlement with time-dependent degree of consolidation
Bearing Capacity (Meyerhof) Python
# General bearing capacity — Meyerhof factors import math def bearing_capacity(c, phi_deg, gamma, Df, B, L): phi = math.radians(phi_deg) Nq = math.exp(math.pi * math.tan(phi)) \ * math.tan(45 + phi_deg/2)**2 Nc = (Nq - 1) / math.tan(phi) Ng = 2 * (Nq + 1) * math.tan(phi) qu = c*Nc + gamma*Df*Nq + 0.5*gamma*B*Ng return qu, qu/3 # ultimate, allowable (FOS=3)
Ultimate and allowable bearing capacity with shape/depth factors
